# Datos de ejemplo

Conjuntos de datos pequeños y de dominio humanístico para las prácticas del curso.

- `fragmento_literario.txt` — fragmento de *Don Quijote de la Mancha* (dominio público),
  útil para conteo de palabras y análisis de texto.
- `fechas_historicas.csv` — pequeña tabla de eventos y años, útil para clasificar por
  siglo y para primeras prácticas con datos tabulares.

Todos son de dominio público o de elaboración propia.
